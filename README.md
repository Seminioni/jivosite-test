
### Подготовка

1. Установить [nvm](https://github.com/creationix/nvm)
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.8/install.sh | bash
```
2. Установить lts-версию node.js
```
nvm install --lts
```

### Установка зависимостей
В корне папки

```
yarn
```
### Запуск дев-режима

```
yarn start
```
### Сбилдить проект

```
yarn build

```
